# TwinBrain 虚拟刺激与动力系统验证模块实现计划书

## 一、目标

在现有训练完成的 TwinBrain 模型基础上，新增一个 **独立的动力系统分析模块**，用于验证模型是否真正学习到了脑网络的动态结构，而不仅仅是进行时间序列拟合。

该模块不参与训练，仅加载训练好的模型进行测试分析。
建议作为 **独立子模块** 实现，以避免干扰主训练流程。

核心目标：

1. 分析模型 **自由动力学（free dynamics）**
2. 检测系统 **吸引子结构**
3. 实现 **虚拟脑刺激实验**
4. 计算 **刺激响应矩阵**
5. 验证模型 **稳定性与动力系统合理性**

```python
def load_trained_model(model_path: str, device: str):
    model = TwinBrain(...)
    state = torch.load(model_path, map_location=device)
```

```python
class BrainDynamicsSimulator:

    def __init__(self, model):
        self.model = model

    def step(self, state, external_input=None):
        """
        单步动力学演化

        x(t+1) = f(x(t), u(t))
        """

        """
        连续动力学模拟
        """
```

要求：

* 所有实验通过该接口进行
* 支持 **外部刺激输入**

# 五、自由动力学实验

文件：

```
experiments/free_dynamics.py
```

目标：

验证系统在 **无输入情况下** 的长期行为。

实现流程：

```
1 采样多个初始状态
2 长时间自由演化
3 记录状态轨迹
```

实现示例：

```python
def run_free_dynamics(simulator, n_init=200, steps=1000):

    trajectories = []

    for i in range(n_init):

        x0 = sample_random_state()

        traj = simulator.rollout(
            x0=x0,
            steps=steps,
            stimulus=None
        )

        trajectories.append(traj)

    return trajectories
```

输出：

```
trajectories.npy
```

# 八、响应矩阵计算

文件：

```
analysis/response_matrix.py
```

目标：

量化 **刺激传播结构**。

定义：

```
R[i,j] = response of node j
         when node i is stimulated
```

计算步骤：

```
for each node i:

    apply stimulus to node i

    measure activity change of all nodes

    store result
```

实现示例：

```python
def compute_response_matrix(simulator, n_nodes):

    R = np.zeros((n_nodes, n_nodes))

    for i in range(n_nodes):

        response = run_stimulation(simulator, node=i)

        R[i] = response

    return R
```

输出：

```
response_matrix.npy
```

# 九、稳定性分析

文件：

```
analysis/stability_analysis.py
```

目标：

验证系统是否存在稳定动力学。

方法：

计算状态变化：

```
Δ(t) = ||x(t) − x(t−1)||
```

判断：

```
Δ → 0  → fixed point
周期变化 → limit cycle
缓慢漂移 → metastable state
```

输出：

```
stability_metrics.json
```

# 十、连续刺激要求（关键设计）

刺激 **必须为连续输入**。

禁止：

```
单时间点脉冲
```

原因：

神经系统对持续输入产生动力学响应。

推荐形式：

```
sinusoidal
square wave
step input
```

刺激函数接口：

```python
class Stimulus:

    def value(self, t):
        pass
```

# 十五、成功判据

若模型学习到真实动力学，应出现：

1. 有限数量吸引子
2. 稳定轨迹
3. 可重复刺激响应
4. 非随机传播结构

若系统仅为回归模型：

通常会出现：

```
random drift
unstable dynamics
无稳定吸引子
```
