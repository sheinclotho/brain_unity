# 更新日志

所有版本更新记录及待优化内容均记录于此。

## [5.0.0] - 2026-02-20

### 新增
- 新增零配置启动脚本 `start.py`，直接运行即可使用，无需任何参数
- 新增 Web 前端 (`web_frontend/`)，支持在浏览器中直接可视化大脑活动，无需安装 Unity
- Web 前端支持：点击脑区施加虚拟刺激、实时显示活动变化、颜色热图
- 新增 `AGENTS.md`，为 AI 编程助手提供协作规范和错误日志
- 新增 `使用说明.md`，面向非技术用户的简明操作指南

### 变更
- **文档整合**：将原有 9 个 Markdown 文件整合为 4 个（规范、使用说明、更新日志、AI 提示词）
- `start.py` 替代原有复杂的 `unity_startup.py` 作为默认启动入口（`unity_startup.py` 仍保留供高级用户使用）
- 默认所有模块启用，无需手动选择
- WebSocket 默认绑定 `127.0.0.1`（本地），提升安全性

### 待优化
- [ ] Unity 3D 场景自动化设置流程仍需手动操作，有待简化
- [ ] Web 前端目前使用简化的球形脑区，待集成真实 FreeSurfer 脑表面数据
- [ ] 添加单元测试覆盖率至 60% 以上
- [ ] 支持 Docker 一键部署

## [4.1.0] - 2024-02-15

### 新增
- Unity 自动化设置工具
- 虚拟刺激 UI
- 文件监控与自动重载
- 一键安装脚本
- 多种刺激模式支持（正弦波、脉冲、斜坡、持续）

### 变更
- 改进 WebSocket 通信稳定性
- JSON 导出格式升级至 v2.0
- 优化 brain_state_exporter 内存使用

### 修复
- Windows 路径处理问题
- Unity 包安装异常
- FreeSurfer 数据加载边界情况

## 迁移指南

### 从 4.x 迁移到 5.0

- 直接运行 `python start.py` 代替原来的 `python unity_startup.py --demo`
- Unity 集成方式不变，后端 WebSocket 接口兼容
- 文档文件名变更，请更新书签
