## 1. 项目概述

### 1.1 核心目标

TwinBrain是一个**基于多模态脑成像数据的数字孪生脑系统**，主要功能包括：
- 大脑活动建模与预测
- Unity 3D实时可视化
- 多模态数据整合（fMRI、EEG、DTI）
- 虚拟刺激模拟

### 1.2 技术栈

| 层级 | 技术 |
|------|------|
| **核心框架** | Python 3.8+ |
| **深度学习** | PyTorch 1.12+ |
| **神经网络架构** | 异构图神经网络 (Heterogeneous GNN) |
| **可视化引擎** | Unity 2019 LTS+ |
| **通信协议** | WebSocket (websockets库) |
| **数据格式** | JSON, OBJ, NIfTI, BIDS |
| **神经数据处理** | nibabel, nilearn, scipy |

##### A. `dynamic_hetero_gnn.py`

**异构图神经网络架构**:

```python
class DynamicHeteroGNN(nn.Module):
    def __init__(self, config):
        super().__init__()
        self.node_embeddings = nn.ModuleDict({
            'region': nn.Linear(in_features, hidden_dim),
            'network': nn.Linear(in_features, hidden_dim)
        })
        
        self.gnn_layers = nn.ModuleList([
            HeteroConv({
                ('region', 'connects', 'region'): GCNConv(...),
                ('region', 'belongs_to', 'network'): SAGEConv(...),
            }) for _ in range(num_layers)
        ])
        
        self.temporal_model = nn.LSTM(hidden_dim, hidden_dim)
        self.attention = MultiHeadAttention(hidden_dim, num_heads)
        self.predictor = nn.Linear(hidden_dim, out_features)
        
    def forward(self, graph_data, edge_index_dict, edge_attr_dict):
        # 1. 节点嵌入
        x_dict = {node_type: self.node_embeddings[node_type](x) 
                  for node_type, x in graph_data.items()}
        
        # 2. 图卷积
        for gnn_layer in self.gnn_layers:
            x_dict = gnn_layer(x_dict, edge_index_dict)
            x_dict = {k: F.relu(v) for k, v in x_dict.items()}
        
        # 3. 时序建模
        x_temporal, (h, c) = self.temporal_model(x_dict['region'])
        
        # 4. 注意力机制
        x_attended = self.attention(x_temporal)
        
        # 5. 预测
        output = self.predictor(x_attended)
        return output
```

**图数据结构**:

##### B. `hetero_trainer_v2.py`

**训练循环**:
```python
class HeteroTrainerV2:
    def train(self, model, data_loader, optimizer, epochs):
        for epoch in range(epochs):
            for batch in data_loader:
                # Forward
                pred = model(batch)
                loss = self.compute_loss(pred, batch.y)
                
                # Backward
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()
                
                # Logging
                self.log_metrics(epoch, loss, pred, batch.y)
                
    def compute_loss(self, pred, target):
        # 多任务损失
        recon_loss = F.mse_loss(pred['reconstruction'], target)
        pred_loss = F.mse_loss(pred['prediction'], target)
        reg_loss = self.l2_regularization()
        
        return recon_loss + 0.5 * pred_loss + 0.01 * reg_loss
```

```csharp
using UnityEngine;
using System.Collections.Generic;
using Newtonsoft.Json;

namespace TwinBrain
{
    public class BrainVisualization : MonoBehaviour
    {
        // 配置参数
        [SerializeField] private string jsonPath = "StreamingAssets/state";
        [SerializeField] private GameObject regionPrefab;
        [SerializeField] private float regionScale = 0.5f;
        [SerializeField] private float activityThreshold = 0.2f;
        
        // 数据容器
        private List<BrainStateData> brainStates;
        private Dictionary<int, GameObject> regionObjects;
        private int currentTimePoint = 0;
        
        void Start()
        {
            LoadBrainData();
            CreateRegionObjects();
            UpdateVisualization();
        }
        
        void Update()
        {
            if (Input.GetKeyDown(KeyCode.Space))
            {
                ToggleAnimation();
            }
            if (Input.GetKeyDown(KeyCode.R))
            {
                ReloadData();
            }
        }
        
        private void LoadBrainData()
        {
            // 从JSON文件加载数据
            string[] jsonFiles = Directory.GetFiles(jsonPath, "*.json");
            brainStates = new List<BrainStateData>();
            
            foreach (string file in jsonFiles)
            {
                string jsonText = File.ReadAllText(file);
                BrainStateData state = JsonConvert.DeserializeObject<BrainStateData>(jsonText);
                brainStates.Add(state);
            }
        }
        
        private void CreateRegionObjects()
        {
            regionObjects = new Dictionary<int, GameObject>();
            
            if (brainStates.Count > 0)
            {
                foreach (var region in brainStates[0].brain_state.regions)
                {
                    GameObject obj = Instantiate(regionPrefab, 
                        new Vector3(region.xyz[0], region.xyz[1], region.xyz[2]), 
                        Quaternion.identity);
                    obj.name = $"Region_{region.id}_{region.label}";
                    obj.transform.localScale = Vector3.one * regionScale;
                    regionObjects[region.id] = obj;
                }
            }
        }
        
        private void UpdateVisualization()
        {
            if (currentTimePoint >= brainStates.Count) return;
            
            var currentState = brainStates[currentTimePoint];
            
            foreach (var region in currentState.brain_state.regions)
            {
                if (regionObjects.ContainsKey(region.id))
                {
                    GameObject obj = regionObjects[region.id];
                    
                    // 根据活动值更新颜色
                    float activity = region.activity;
                    Color color = GetActivityColor(activity);
                    obj.GetComponent<Renderer>().material.color = color;
                    
                    // 根据活动值更新大小（可选）
                    if (activity > activityThreshold)
                    {
                        float scale = regionScale * (1 + activity);
                        obj.transform.localScale = Vector3.one * scale;
                    }
                }
            }
        }
        
        private Color GetActivityColor(float activity)
        {
            // 低活动=蓝色，高活动=红色
            return Color.Lerp(Color.blue, Color.red, activity);
        }
    }
}
```

## 5. 关键算法实现

### 5.1 球体网格生成算法

```python
def generate_sphere_vertices(center, radius, resolution):
    """
    使用球坐标系统生成球体网格
    
    球坐标公式:
        x = r * sin(θ) * cos(φ)
        y = r * sin(θ) * sin(φ)
        z = r * cos(θ)
    
    其中:
        θ (theta) = 纬度角 [0, π]
        φ (phi) = 经度角 [0, 2π]
    """
    vertices = []
    normals = []
    faces = []
    
    # 生成顶点
    for i in range(resolution + 1):
        lat = np.pi * i / resolution  # 纬度
        
        for j in range(resolution + 1):
            lon = 2 * np.pi * j / resolution  # 经度
            
            # 单位球面坐标
            x = np.sin(lat) * np.cos(lon)
            y = np.sin(lat) * np.sin(lon)
            z = np.cos(lat)
            
            # 缩放并平移
            vertex = center + radius * np.array([x, y, z])
            vertices.append(vertex)
            
            # 法线（指向球外）
            normals.append([x, y, z])
    
    # 生成面（四边形分为两个三角形）
    for i in range(resolution):
        for j in range(resolution):
            # 四个顶点索引
            v1 = i * (resolution + 1) + j
            v2 = v1 + 1
            v3 = (i + 1) * (resolution + 1) + j
            v4 = v3 + 1
            
            # 两个三角形
            faces.append((v1, v3, v2))
            faces.append((v2, v3, v4))
    
    return np.array(vertices), np.array(normals), faces
```

### 5.2 异构图消息传递

```python
def hetero_message_passing(x_dict, edge_index_dict, edge_attr_dict):
    """
    异构图神经网络的消息传递
    
    对于每种边类型 (src_type, edge_type, dst_type):
        1. 聚合邻居节点的特征
        2. 应用边类型特定的转换
        3. 更新目标节点特征
    """
    out_dict = {}
    
    for edge_type, edge_index in edge_index_dict.items():
        src_type, relation, dst_type = edge_type
        
        # 源节点特征
        x_src = x_dict[src_type]
        
        # 边特征（如果有）
        edge_attr = edge_attr_dict.get(edge_type, None)
        
        # 消息计算
        messages = compute_messages(x_src, edge_index, edge_attr)
        
        # 聚合
        aggregated = aggregate_messages(messages, edge_index[1])
        
        # 更新
        if dst_type not in out_dict:
            out_dict[dst_type] = []
        out_dict[dst_type].append(aggregated)
    
    # 合并来自不同边类型的消息
    for node_type in out_dict:
        out_dict[node_type] = sum(out_dict[node_type])
    
    return out_dict
```

### 5.3 多步预测算法

```python
def multi_step_prediction(model, initial_state, n_steps):
    """
    自回归多步预测
    
    t=0: initial_state
    t=1: model(initial_state)
    t=2: model(predicted_t1)
    ...
    t=n: model(predicted_t{n-1})
    """
    predictions = [initial_state]
    current_state = initial_state
    
    for step in range(n_steps):
        with torch.no_grad():
            # 预测下一步
            next_state = model(current_state)
            predictions.append(next_state)
            
            # 更新当前状态
            current_state = next_state
    
    return torch.stack(predictions, dim=1)  # [batch, n_steps+1, features]
```

```python
def process_large_dataset(data_path, chunk_size=1000):
```
