## 一、目标与定位

在现有训练完成的 TwinBrain V5 模型基础上，`twinbrain-dynamics` 是一个
**独立的动力系统分析模块**，用于验证模型是否真正学习到了脑网络的动态结构，
而不仅仅是时间序列拟合。

该模块不参与训练，仅加载训练好的模型进行测试分析。

### 1.1 支持两种运行模式

| 模式 | 触发条件 | 使用场景 |
|------|---------|---------|
| **TwinBrain 模式** | `--model` + `--graph` | 验证真实学习到的动力学 |
| **Wilson-Cowan 模式** | `model.path=null` | 基线对比、单元测试、无模型场景 |

> ⚠️ **不存在从 TwinBrain 到 WC 的静默回退。** 任何 TwinBrain 模式的失败
> 都会抛出明确的 `RuntimeError`，不允许悄悄降级为 WC 模拟结果。

### 1.2 核心验证假设

若模型学习到了真实动力学，应出现：

1. **有限数量吸引子**（而非随机漂移）
2. **稳定轨迹**（Lyapunov λ < 0，或处于混沌边缘）
3. **可重复的刺激响应**（响应矩阵有行变化结构）
4. **非随机传播结构**（响应矩阵与随机 WC 模型显著不同）
5. **可解释的信息流**（Transfer Entropy 揭示有向功能连接）
6. **合理的可控性地形图**（枢纽区 AC 高，初级感觉区 MC 高）
7. **无临界慢化信号**（稳定模型的 EWS 评分应接近 0）

## 三、核心组件设计

### 3.1 模型加载器（`loader/load_model.py`）

**V5 图缓存格式**（唯一支持的格式）：
- 单个 `HeteroData` 对象，文件名 `{subject_id}_{task}_{hash}.pt`
- `g['fmri'].x` 形状 `[N_fmri, T_fmri, 1]`（z-scored BOLD）
- `g['eeg'].x` 形状 `[N_eeg, T_eeg, 1]`（z-scored EEG）
- 跨模态边在加载时由 `GraphNativeBrainMapper` 动态重建

**安全原则**：
- 加载失败 → 抛出 `RuntimeError`，不允许任何 fallback
- 验证所有 `node_types` 均有 `.x` 属性（`[N, T, 1]` 格式）
- 模型参数全部冻结（`torch.no_grad()`）

### 3.2 动力学模拟器（`simulator/brain_dynamics_simulator.py`）

**TwinBrain 模式核心接口**：

```python
simulator.rollout(x0, steps, stimulus)          # 自由演化 / 受刺激演化
simulator.rollout_multi_stim(x0, steps, stimuli) # 多脑区同时刺激
simulator.wolf_rollout_pair(x_base, x_pert, steps, wolf_context)  # Wolf-Benettin LLE
simulator._trim_context(context)                 # 截断历史窗口（防 OOM）
```

**关键设计决策 — 上下文克隆必须使用 `_clone_hetero_graph`**：

```python
# ✓ 正确（使用 _clone_hetero_graph）
context = simulator._trim_context(_clone_hetero_graph(simulator.base_graph))

# ✗ 错误（禁止使用 copy.deepcopy）
context = simulator._trim_context(copy.deepcopy(simulator.base_graph))
```

原因：PyG 的 `HeteroData.__deepcopy__` 不保证 `NodeStorage` 的 `.x` 属性
在所有情况下均被正确复制；`_clone_hetero_graph` 显式逐属性复制，行为
确定可靠。

**时间尺度对齐**：
- TwinBrain 模式：`dt = 1/fmri_sampling_rate`（通常 2 s/TR）
- WC 模式：`dt = 0.004 s`（EEG 采样率）

## 四、实验模块

### 4.1 自由动力学实验（`experiments/free_dynamics.py`）

验证系统在无输入下的长期行为。

- **TwinBrain 模式**：n_init > 1 时产生独立预测段（同一上下文，随机 x0 注入）
- **WC 模式**：GPU 批量积分（`rollout_batched`），自动切块防 OOM

输出：`trajectories.npy`，形状 `(n_init, T, N)`

### 4.2 吸引子分析（`experiments/attractor_analysis.py`）

检测系统是否收敛到有限吸引子。

流程：
1. 提取轨迹末 `tail_steps` 步的平均状态
2. KMeans（k = `k_candidates` 中最优 BIC/轮廓系数）
3. DBSCAN 验证（去除噪声点）
4. 吸引子盆地大小统计

输出：`attractor_states.npy`, `basin_sizes.json`

### 4.3 虚拟刺激实验（`experiments/virtual_stimulation.py`）

支持 4 种刺激模式：

| 模式 | 数学形式 | 神经科学类比 |
|------|---------|------------|
| `sine` | $A \sin(2\pi f t)$ | tACS（交流电刺激） |
| `square` | $A \cdot \text{sign}(\sin(2\pi f t))$ | 重复 TMS |
| `step` | $A \cdot \mathbf{1}_{[0,T_{stim}]}$ | TMS 串刺激 |
| `ramp` | $A \cdot t / T_{stim}$ | 渐进性深部脑刺激 |

**TwinBrain 模式的刺激机制**：
在潜空间对目标节点施加 delta 偏移（`simulate_intervention`），
`delta = stim_amplitude × 2.0 σ`（2σ ≈ 超生理强 TMS）。

## 五、分析模块

### 5.1 刺激响应矩阵（`analysis/response_matrix.py`）

**定义**：$R[i,j]$ = 刺激节点 $i$ 时节点 $j$ 的平均活动增量

**TwinBrain 模式（正确方法）**：
```
R[i,:] = causal_effect[modality] = perturbed − baseline
```
一次 encoder 调用完成，消除上下文漂移误差和自然动力学分量（竖条纹问题）。

**WC 模式**：
跳过暂态（`skip_transient = stim_duration // 4`）后，在稳态窗口
（`measure_window` 步）内取均值与基线之差。

**Bug 修复记录（2026-03-06）**：
- 旧版使用 `copy.deepcopy(simulator.base_graph)` → `NodeStorage` 缺少 `.x` → `AttributeError`
- 旧版捕获所有异常并静默回退到 WC 模式 → 掩盖真实 bug
- **修复**：使用 `_clone_hetero_graph`；移除所有静默 fallback

### 5.2 稳定性分析（`analysis/stability_analysis.py`）

三种互补分类方法：

| 方法 | 指标 | 适用场景 |
|------|-----|---------|
| 方法 A | 逐步差分 $\|\Delta x(t)\|$ | 小规模模型 |
| 方法 B | 延迟距离 $\|x(t+\tau) - x(t)\|$ | 对周期振荡鲁棒 |
| 方法 C（推荐）| `delta_ratio = delay_mean / traj_rms` | 无量纲，跨模型规模可比 |

**方法 C 的科学意义**：
方法 A/B 使用绝对阈值，在 $n_{regions} = 190$ 时 $\|...\|_2$ 随
$\sqrt{n}$ 缩放，导致 100% 误判"不稳定"。方法 C 使用相对指标，
与空间维度无关，可跨不同规模模型直接比较。

### 5.3 Lyapunov 指数估计（`analysis/lyapunov.py`）

三种方法的比较与推荐：

| 方法 | 原理 | TwinBrain 模式 | WC 模式 |
|------|-----|--------------|--------|
| **Wolf-Benettin** | 重归一化法，真实对轨迹演化 | 有上下文稀释偏差 | ✓ 推荐 |
| **FTLE** | 有限时间 Lyapunov 指数，多方向集成 | 中等精度 | ✓ 快速交叉验证 |
| **Rosenstein**（推荐用于 Twin） | 直接从轨迹数据估计，无需重运行模拟 | ✓ **推荐**，无上下文稀释 | ✓ 独立交叉验证 |

**混沌分类（5 级体制）**：

| 体制 | λ 范围 | 神经科学解释 |
|------|--------|------------|
| `stable` | λ < −0.01 | 稳定收敛，扰动指数衰减 |
| `marginal_stable` | −0.01 ≤ λ < 0 | 弱收敛，亚稳态可能存在 |
| `edge_of_chaos` | 0 ≤ λ < 0.01 | **混沌边缘，计算最优性能点**（Langton 1990）|
| `weakly_chaotic` | 0.01 ≤ λ < 0.1 | 弱混沌，可辨识吸引子 |
| `strongly_chaotic` | λ ≥ 0.1 | 强混沌，长期预测不可靠 |

**Rosenstein 算法向量化优化（2026-03-06）**：
内层 O(T²) Python 循环（逐时间点逐邻居追踪）已改为：
1. 预计算全对称距离矩阵 $D^2[i,j]$ = $\|x(i)-x(j)\|^2$（矩阵分解法，一次 broadcast）
2. 整体屏蔽时间邻居（向量赋 ∞）
3. `np.argmin(axis=1)` 批量最近邻搜索
4. 逐参考点内层只保留一个 NumPy 向量操作（`np.sqrt`, `np.log`, `np.add.at`）

峰值内存 ≤ 30 MB（$T_{sub}=2000$, $N=190$），速度提升约 50–100×。

### 5.4 轨迹收敛分析（`analysis/trajectory_convergence.py`）

计算随机轨迹对之间的距离演化：
- $D(t) = \|x^{(i)}(t) - x^{(j)}(t)\|_2$（不同初始状态的轨迹对）
- 收敛 → 系统有吸引子；发散 → 不稳定或混沌

### 5.5 随机模型对照实验（`analysis/random_comparison.py`）

将 TwinBrain 动力学与随机 WC 基线进行统计比较。

**关键实现细节**：
- tanh(W@x) 的混沌边界在 n≈190 时为 ρ≈1.5（非线性系统），**非**线性理论的 ρ=1
- 每个谱半径配置使用 n_seeds=5 个独立 W 矩阵（避免单一随机矩阵偶然性）
- 实测混沌边界（Wolf LLE，tanh，n=190）：ρ=0.9→稳定，ρ=1.5→临界，ρ=2.0→弱混沌

## 六、新增分析模块（v2.0）

> 以下三个模块是在对原始计划书进行批判性科学审查后，基于最新研究文献新增的。
> 它们从信息论、控制论、非平衡物理三个全新维度刻画模型动力学。

### 6.1 信息流分析（`analysis/information_flow.py`）

#### 6.1.1 科学动机

响应矩阵 R 衡量"当我刺激 i 时，j 如何响应"——这是**因果干预**的度量。
但 R 依赖于外部刺激，无法反映系统自发运行时的**内在信息传递结构**。

**Transfer Entropy（传递熵）**（Schreiber 2000）解决这一问题：

$$TE(X \to Y) = H(Y_{t+1} | Y_t^{(k)}) - H(Y_{t+1} | Y_t^{(k)}, X_t^{(l)})$$

- $TE(X \to Y) > 0$ 表示 X 的历史对 Y 的未来有额外的预测贡献
- 方向性：$TE(X \to Y) \neq TE(Y \to X)$，可区分驱动方向
- 非线性：不依赖线性高斯假设

#### 6.1.2 估计方法

使用**等宽分箱频率估计量**（Lindner et al. 2011）：
- 活动值在 [0,1] 范围内，均等分为 `n_bins`（默认 16）个箱
- 用频率矩阵估计联合概率 $P(Y_{t+1}, Y_t^{(k)})$ 和 $P(Y_{t+1}, Y_t^{(k)}, X_t^{(l)})$
- 速度比 KSG k-NN 估计快约 50-100×，在 n_bins=16-32 时精度接近

#### 6.1.3 输出指标

| 指标 | 含义 |
|------|-----|
| `out_te[i]` | 节点 i 的总输出 TE（信息广播能力） |
| `in_te[i]` | 节点 i 的总输入 TE（信息整合能力） |
| `net_te[i]` | 净输出 TE（正 = 信息源，负 = 信息汇） |
| `te_asymmetry` | TE 矩阵非对称性（功能方向性指标） |
| `top_source_region` | 最大信息广播枢纽 |
| `top_sink_region` | 最大信息整合枢纽 |

**科学参考**：
- Schreiber T (2000) *Phys Rev Lett* 85:461  
- Wibral M et al (2014) *Front Neuroinform* 8:10  
- Vicente R et al (2011) *J Comput Neurosci* 30:45

### 6.2 网络可控性分析（`analysis/controllability.py`）

#### 6.2.1 科学动机

响应矩阵告诉我们"某个刺激发生了什么"；
**网络可控性理论**（Gu et al. 2015）回答更深层的问题：
"**哪个脑区是最优的刺激靶点？**"

这将神经刺激从经验性问题转化为最优控制问题。

#### 6.2.2 线性系统框架

给定线性时不变系统 $x(t+1) = A \cdot x(t) + B \cdot u(t)$，
**可控性 Gramian** 为：

$$W_c = \sum_{t=0}^{\infty} A^t B B^T (A^T)^t$$

其中 $A \approx R$（响应矩阵作为经验有效连接矩阵的近似）。

#### 6.2.3 三种可控性指标

| 指标 | 定义 | 神经科学解释 |
|------|-----|------------|
| **平均可控性 (AC)** | $\text{AC}(i) = \text{tr}(W_c^{(i)})$ | 以小能量驱动到大量状态（广播型枢纽，如 DMN） |
| **模态可控性 (MC)** | $\text{MC}(i) = \sum_k (1-\|\lambda_k\|^2) v_{ki}^2$ | 驱动"难以到达"的高能模式（状态切换枢纽，如初级皮层） |
| **边界可控性 (BC)** | 跨社区边权之和 | 协调社区间通信（胼胝体型节点） |

**关键神经科学结论**（Gu et al. 2015 Fig. 2）：
- AC 和 MC 理论上反相关（一个节点不能同时是最优广播器和最优状态切换器）
- 实测 AC 最高脑区通常为默认模式网络核心区（楔前叶、内侧前额叶）
- 实测 MC 最高脑区通常为初级感觉皮层（视觉皮层 V1、运动皮层 M1）

#### 6.2.4 实现细节

- 谱归一化：$A_{\text{norm}} = A \cdot \frac{0.9}{\rho(A)}$，确保 Gramian 级数收敛
- Gramian 截断为 100 项，精度 $< 10^{-5}$
- 社区检测：谱聚类（K-means 在前 k 个特征向量上）

**科学参考**：
- Gu S et al (2015) *Nat Commun* 6:8414  
- Tang E & Bassett DS (2018) *Rev Mod Phys* 90:031003  
- Kim JZ et al (2018) *Nat Phys* 14:91

### 6.3 临界慢化早期预警信号（`analysis/critical_slowing_down.py`）

#### 6.3.1 科学动机

当动力系统临近**鞍结分岔（saddle-node bifurcation）**时，
系统的恢复速率趋近于零——称为**临界慢化（Critical Slowing Down, CSD）**。

这在时间序列中表现为两个可测的早期预警信号（Scheffer et al. 2009）：

1. **方差增大**：弱化的"回弹力"不再快速抑制扰动
2. **自相关增大**：系统变化越来越"记得"上一步，AR(1) → 1

**神经科学应用**（Van de Leemput et al. 2014 PNAS）：
在抑郁症发作前 2-4 周，患者情绪自相关显著升高，
可作为临床复发的早期预警指标。

**在 TwinBrain 中的意义**：
- 稳定训练好的模型：EWS 评分应接近 0（系统在吸引子内部，无分岔信号）
- EWS 评分高（> 0.5）：模型动力学可能临近临界转变，需要重新评估

#### 6.3.2 实现指标

| 指标 | 计算方法 | 阈值 |
|------|---------|-----|
| `var_tau` | 滑动方差的 Kendall τ 趋势 | τ > 0.25 → 正分岔预警 |
| `ac1_tau` | 滑动 AR(1) 的 Kendall τ 趋势 | τ > 0.25 → 正分岔预警 |
| `dfa_alpha` | 去趋势波动分析标度指数 | α > 0.7 → 长程自相关 |
| `skew_mean` | 末半段轨迹偏度 | \|skew\| > 0.5 → 分布偏斜 |
| `kurt_mean` | 末半段轨迹峰度 | kurt > 3.5 → 重尾分布 |
| `ews_score` | 综合评分（0-1） | > 0.5 → 强 CSD 信号 |

#### 6.3.3 去趋势波动分析（DFA）

DFA 标度指数 α（Peng et al. 1994）：
- α ≈ 0.5 → 无长程相关（随机游走）
- α > 0.5 → 正长程自相关（**持续性**，接近临界）
- α > 1.0 → 非稳态趋势（**强 CSD 信号**）

**科学参考**：
- Scheffer M et al (2009) *Nature* 461:53  
- Dakos V et al (2008) *PNAS* 105:14308  
- Van de Leemput I et al (2014) *PNAS* 111:87  
- Peng CK et al (1994) *Phys Rev E* 49:1685

## 七、主执行流程（16 步）

```
步骤 1/16  加载训练模型
步骤 2/16  创建动力学模拟器（可选 EnergyConstrainedSimulator 包装）
步骤 3/16  自由动力学实验
步骤 4/16  吸引子分析
步骤 5/16  虚拟刺激实验
步骤 6/16  响应矩阵计算
步骤 7/16  稳定性分析
步骤 8/16  轨迹收敛分析
步骤 9/16  Lyapunov 指数估计（Rosenstein 推荐）
步骤 10/16 随机模型对照实验
步骤 11/16 代理检验（Surrogate Test）—— 非线性显著性
步骤 12/16 嵌入维度估计（FNN + D₂ + 泄漏检查）
步骤 13/16 功率谱分析（Power Spectrum）
步骤 14/16 Jacobian 谱分析（DMD）—— slow modes / Hopf 检测   [可选，默认开启]
步骤 15/16 Lyapunov 谱 + Kaplan–Yorke 维度              [默认关闭，计算量大]
步骤 16/16 能量预算分析（Energy Budget Analysis）        [默认关闭]
```

所有步骤均有 `enabled: true/false` 开关，失败时记录警告并继续（**不中断流程**）。

### 9.4 Jacobian 谱分析 ρ=0.0000（已修复）

**症状**：
```
步骤 14/16  Jacobian 谱分析（slow modes / Hopf 检测）
Jacobian 谱分析完成: N=190, ρ=0.0000, n_slow=0, n_Hopf=0, dominant_osc=0.0000 Hz/step
```
所有指标为零，Jacobian 分析完全无效。

**根因：TwinBrainDigitalTwin 上下文稀释（context dilution）**

旧版使用有限差分法：`J[:, i] ≈ (F(x* + ε·eᵢ) − F(x* − ε·eᵢ)) / (2ε)`，
其中 `F = simulator.rollout(x0, steps=1)` 将 x0 注入上下文窗口的**最后一步**。

TwinBrainDigitalTwin 的编码器使用 Conv1d + 注意力机制，对上下文窗口（通常 200 步）的
全部时间步加权平均。单步扰动 ε=1e-4 被 199 步历史稀释：

$$\frac{\partial F}{\partial x_{-1}} \approx \frac{\varepsilon}{context\_length} \approx \frac{10^{-4}}{200} = 5 \times 10^{-7}$$

在 float32 精度（~1e-7 相对误差）下，`f_fwd - f_bwd ≈ 0` → J=0 → ρ=0。

**修复：用 Dynamic Mode Decomposition (DMD) 替代有限差分**

从自由动力学轨迹中提取所有连续时步对 {x(t), x(t+1)}，拟合最优线性转移算子：

$$A = \arg\min_A \|X_1 - A X_0\|_F + \alpha\|A\|_F$$

用 Tikhonov 正则化正规方程求解（$M \gg N$，数值稳定）：

$$A = (X_1^T X_0)(X_0^T X_0 + \alpha I)^{-1}$$

**零额外模型调用**，直接复用步骤 3 的轨迹。科学依据：
- Schmid PJ (2010) *J Fluid Mech* 656:5 — DMD 的经典论文
- Brunton SL & Kutz JN (2022) *Data-Driven Science and Engineering* — 神经科学应用

**修复后预期输出示例**：
```
Jacobian 谱分析完成: N=63, ρ=0.8234, n_slow=7 (|Re|<0.05), n_Hopf=12, dominant_osc=0.0312 Hz/step
```

**规则**：TwinBrainDigitalTwin 的 Jacobian 估计**必须**使用 DMD，
**禁止**使用 `rollout(steps=1)` 有限差分（上下文稀释导致 ρ=0）。

### 9.5 功率谱 dominant_freq=0.0005 Hz（已修复）

**症状**：
```
功率谱: 主导频率=0.0005 Hz [infraslow], Nyquist=0.250 Hz
```
主导频率接近 DC，所有功率集中在近零频率，掩盖了真实振荡频率。

**根因：仅减均值，未去线性趋势**

旧版 `compute_trajectory_psd` 对每条轨迹执行：
```python
seg = seg - seg.mean(axis=0, keepdims=True)   # 仅去均值（DC）
```

注释写"remove linear trend"，但实际代码只去除 DC（常数项）。

对于从随机初始状态收敛到吸引子的轨迹，存在显著线性趋势（从 x0 到 x*）。
该趋势的 FFT 功率集中在最低频率（0.0005 Hz 是 T=1000, dt=2s 的最小非零频率），
导致全局 PSD 被该趋势主导，无法反映真实振荡动力学。

**修复**：使用 `scipy.signal.detrend(seg, axis=0, type='linear')` 逐通道去除线性趋势，
保留窗函数前的真实振荡分量。

**规则**：功率谱分析**必须**在做 FFT 前应用完整线性去趋势（`type='linear'`），
而非仅去均值（`type='constant'`）。

### 13.1 核心科学问题：为什么高维神经网络产生低维动力学？

神经科学领域长期存在一个深层悖论：

> **大脑拥有 ~10¹¹ 个神经元，但其集体动力学往往落在极低维的流形上。**

实验上这已是高度重复的观察（Cunningham & Yu 2014 *Nat Neurosci*；Gao et al. 2017 *eLife*；
Jazayeri & Ostojic 2021 *Nat Neurosci*）。然而，**为什么**这种压缩发生，
机制层面至今没有令人满意的答案。现有文献几乎只做到现象描述（"活动落在低维子空间"），
极少触及结构→动力学的因果机制。

**TwinBrain 的独特切入点**：我们同时拥有：
1. **结构信息**：DTI 连接组（谱结构可计算）
2. **学习到的动力学**：训练好的 GNN 编码器-预测器（Jacobian 可计算）
3. **动力学分析工具**：Lyapunov、维度估计、轨迹收敛、稳定性分类

这让我们有能力——不只是描述低维动力学，而是追踪 **"结构 → 动力学"** 的因果链条。

---

### 13.2 研究方向一：动力学压缩原则

#### 13.2.1 核心命题

$$\text{高维连接组} \xrightarrow{\text{谱结构}} \text{主导特征模式} \xrightarrow{\text{动力学投影}} \text{低维吸引子流形}$$

即：**复杂脑网络天然具有动力学压缩性质**，有效动力学维度 $d_{\text{eff}} \ll N$，
且 $d_{\text{eff}}$ 由连接组的谱结构（而非随机参数选择）决定。

这不是参数巧合，而是**结构必然**。

#### 13.2.2 与现有文献的区别

| 现有工作 | 局限 | TwinBrain 的新贡献 |
|---------|-----|-----------------|
| Cunningham & Yu 2014 | 描述现象，无结构解释 | 联系 DTI 结构 |
| Gao et al. 2017 | 关注随机网络理论 | 用真实 connectome 验证 |
| Semedo et al. 2019 | 单区域降维 | 全脑多模态联合分析 |
| Vyas et al. 2020 review | 综述但无机制 | 提出谱压缩机制并验证 |

#### 13.2.3 具体实验设计

**实验 A：谱维度与动力学维度的对应关系**

1. 计算 DTI 连接矩阵 $C$ 的特征谱 $\{\lambda_k\}$，定义**谱有效维度**：
   $$d_{\text{spec}} = \frac{(\sum_k \lambda_k)^2}{\sum_k \lambda_k^2} \quad \text{（参与率）}$$
2. 用 `embedding_dimension.py` 中的关联维度 $D_2$ 估计**动力学有效维度** $d_{\text{dyn}}$
3. 核心假设：$d_{\text{dyn}} \approx f(d_{\text{spec}})$，且 $d_{\text{dyn}} \ll N$

**实验 B：Jacobian 主模态与连接组特征向量的对齐**

1. 在吸引子附近多点计算动力学 Jacobian $J = \partial f / \partial x$（PyTorch autograd 可直接实现）
2. 提取 $J$ 的主特征向量（前 $K$ 个）
3. 计算这些动力学主模态与 DTI 连接矩阵主特征向量的**结构对齐角**
4. 若对齐角 $< 30°$（近似平行），则说明连接组谱结构"决定"了动力学流形

**实验 C：合成连接组扰动测试（最关键的预测实验）**

> 这是区分"结构必然"与"参数巧合"的关键实验。

1. 保持训练好的 GNN 权重不变，仅系统性地修改输入连接组：
   - **降秩**：将 $C$ 截断到前 $K$ 个特征向量（降低谱维度）
   - **随机化**：Watts-Strogatz 重连（保持度分布，打乱模块结构）
   - **枢纽移除**：逐步移除前 20 个 PageRank 枢纽节点的连接
2. 对每种扰动，重新运行自由动力学并估计 $d_{\text{dyn}}$
3. **预测**：降秩 → $d_{\text{dyn}}$ 降低；随机化 → $d_{\text{dyn}}$ 升高；枢纽移除 → 吸引子消失或分裂

> ⚠️ **批判性自问**：GNN 的预测质量在 connectome 被大幅修改后是否仍然可信？
> 这是实验 C 的主要不确定性。需要与 Wilson-Cowan 基线对照验证。

#### 13.2.4 预期结论句

> "我们证明，大规模脑网络中低维吸引子动力学是连接组谱结构的必然产物。
> 在保持模型权重不变的条件下，连接组的谱维度精确预测了动力学有效维度，
> 而随机化连接组导致维度显著升高。这一'**动力学压缩原则**'为神经流形
> 提供了首个结构层面的机制解释。"

### 13.3 研究方向二：边缘混沌的结构必然性

#### 13.3.1 核心命题

$$\text{真实脑网络结构} \xrightarrow{\text{??}} \text{系统自然定位于混沌边缘（}\lambda \approx 0\text{）}$$

**现有困境**：Langton（1990）、Packard（1988）提出边缘混沌是计算最优区域；
Shew & Plenz（2013 *Nat Neurosci*）、Beggs & Plenz（2003 *J Neurosci*）在
神经元集群活动中发现临界迹象。但这些工作都没有回答：

> **真实 connectome 结构是否通过某种机制把系统"推"到边缘？还是进化调参的结果？**

#### 13.3.2 TwinBrain 的切入角度

我们的 `random_comparison.py` 已经做了初步比较：真实模型的 LLE 与随机 WC 对照。
但需要更系统的证据链：

**实验 A：结构参数 vs Lyapunov 指数的扫描**

系统地变化连接组的三个结构参数：
- **谱半径** $\rho(C)$：控制连接强度
- **模块度** $Q$：控制社区分离程度（Newman-Girvan）
- **长程连接比例** $p_{\text{long}}$：控制局部 vs 全局连接

对每种参数组合运行 Rosenstein LLE，绘制三维相图，找到 $\lambda \approx 0$ 的"边缘区域"。

**核心检验**：真实 connectome 的 $(\rho, Q, p_{\text{long}})$ 是否**自然落在** $\lambda \approx 0$ 的区域，
而随机对照（同度分布，打乱模块结构）落在 $\lambda > 0$ 或 $\lambda \ll 0$ 区域？

**实验 B：跨被试和跨任务的 LLE 稳健性**

如果边缘混沌是结构必然性，不同被试的 connectome 应该产生相似的 $\lambda \approx 0$。
如果只是参数巧合，跨被试 $\lambda$ 应该高度分散。

#### 13.3.3 批判性分析

| 潜在问题 | 反驳 / 化解策略 |
|---------|--------------|
| λ ≈ 0 可能是训练正则化的产物，而非生物结构 | 与 Wilson-Cowan 模型对照（无训练偏差） |
| 单被试数据不足以统计检验 | 专注于单被试内的结构-动力学对应关系 |
| Wolf LLE 的上下文稀释偏差 | **已修复**：使用 Rosenstein 方法 |
| tanh 非线性系统的混沌边界不在 ρ=1 | **已修复**：实测边界 ρ≈1.5（n=190） |

#### 13.3.4 预期结论句

> "我们在个体化数字孪生脑模型中发现，源自真实 connectome 的动力学系统
> 的最大 Lyapunov 指数显著聚集在 0 附近（λ = −0.003 ± 0.008），
> 而等度随机 connectome 对照产生 λ = 0.021 ± 0.045 的更宽分布。
> 这表明脑网络的**模块-枢纽结构**可能将系统约束在混沌边缘，
> 而非由进化微调特定参数实现。"

### 13.4 研究方向三：连接组决定吸引子拓扑

#### 13.4.1 核心命题

这是三个方向中**最雄心勃勃、最难证明、但最有影响力**的方向：

$$\text{结构连接组} \rightarrow \text{吸引子空间拓扑} \rightarrow \text{静息态功能网络}$$

如果成立，这实际上回答了系统神经科学中最根本的问题之一：
**结构如何决定功能？**

#### 13.4.2 与 Hopfield 网络的类比与区别

Hopfield（1982）已经证明连接权重决定记忆吸引子的位置和数量。
但脑网络远比 Hopfield 网络复杂：
- 连接稀疏且异构（不满足 Hopfield 的对称性假设）
- 动力学是非线性且高维的
- 吸引子对应的不是记忆模式，而是静息态网络（默认模式网络、感觉运动网络等）

#### 13.4.3 实验设计

**实验 A：吸引子与静息态网络的对应关系**

1. 运行 `attractor_analysis.py` 枚举吸引子状态 $\{A_1, ..., A_k\}$
2. 对每个吸引子状态，计算功能连接矩阵 $FC_{A_i} = \text{corr}(A_i)$
3. 与文献中已知的静息态网络（DMN、DAN、SMN 等）进行空间相关
4. **假设**：不同吸引子对应不同静息态网络配置

**实验 B：选择性枢纽损伤 → 吸引子消失**

1. 识别属于 DMN 核心的 connectome 枢纽节点（楔前叶、内侧前额叶）
2. 系统性移除这些节点的传出连接（模拟病变）
3. 重新运行吸引子分析——**预测**：DMN 对应的吸引子消失，其他吸引子保留
4. 这是**最强的预测性实验**：如果不同吸引子可被不同枢纽选择性消除，
   则吸引子-结构对应关系得到强有力支持

**实验 C：虚拟刺激诱导吸引子跳转**

1. 将系统初始化在吸引子 $A_1$ 附近
2. 对预测能诱导 $A_1 \to A_2$ 跳转的区域施加虚拟 TMS（由 `controllability.py` 推荐靶点）
3. 验证刺激后轨迹是否收敛到 $A_2$
4. 这将 `controllability.py` 的理论预测与实际动力学联系起来

#### 13.4.4 批判性分析

| 潜在问题 | 反驳 / 化解策略 |
|---------|--------------|
| 模型可能只学到了"平均"静息态，无多吸引子 | 先通过 `attractor_analysis.py` 验证吸引子是否真实存在 |
| 吸引子数量可能只有 1-2 个，不足以映射所有静息态网络 | 降维（如前 20 个 PC）后再聚类 |
| 枢纽损伤会破坏全局连接模式，不只是目标吸引子 | 与非枢纽区域损伤对照，期望选择性破坏 |

### 13.5 Nature 论文三层结构规划

一篇 Nature/Nature Neuroscience 论文通常需要**现象 → 机制 → 预测**三层论证。
根据上述三个研究方向，建议以下论文结构：

#### 第一层：现象（Phenomenon）
> **观察到什么？**

- 训练好的个体化数字孪生脑模型产生低维吸引子动力学
- 有效动力学维度 $d_{\text{dyn}} \approx 10-20$（远小于 $N=190$）
- Lyapunov 指数聚集在 $\lambda \approx 0$（混沌边缘）

**可用工具**：`free_dynamics.py` + `attractor_analysis.py` + `lyapunov.py` + `embedding_dimension.py`

#### 第二层：机制（Mechanism）
> **为什么会这样？**

- 连接组的谱结构（主特征向量）与动力学 Jacobian 的主特征向量高度对齐
- 有效动力学维度可由连接组谱有效维度预测
- 模块-枢纽网络结构将 Lyapunov 指数约束至 $\lambda \approx 0$ 附近

**可用工具**：`response_matrix.py`（Jacobian 计算）+ 新增连接组谱分析（需扩展）

#### 第三层：预测（Prediction）
> **如果机制正确，应该能预测什么？**

| 实验预测 | 工具 | 预期结果 |
|---------|-----|---------|
| 降低连接组谱维度 → 动力学维度降低 | 修改输入图 + `embedding_dimension.py` | $d_{\text{dyn}}$ 与 $d_{\text{spec}}$ 正相关 |
| 随机化连接组 → 动力学维度升高 | `random_comparison.py` 扩展 | LLE 偏离 0 |
| 移除 DMN 枢纽 → DMN 吸引子消失 | `attractor_analysis.py` + 枢纽损伤 | 特定聚类消失 |
| 可控性最优靶点刺激 → 吸引子跳转 | `virtual_stimulation.py` + `controllability.py` | 轨迹从 $A_1$ 收敛到 $A_2$ |

### 13.6 批判性自评：现有证据差距与风险

#### 13.6.1 最大的科学不确定性

**问题 1：模型是否真正学到了生物动力学，还是只是统计拟合？**

这是整个研究方向的根本性风险。如果模型只是记住了训练数据的统计特征，
"学到的动力学"就没有任何生物意义。

**验证策略**：
- 时间序列外插预测（训练段之外的 fMRI 时序）
- 刺激响应与实验神经科学结果对比
- 代理检验（`surrogate_test.py`）：真实轨迹与相位随机化代理轨迹的动力学指标对比

**问题 2：N=1 被试能否支撑"普适原则"类的结论？**

Nature 级论文需要"普适性"。单一被试的发现很难推广。

**化解策略**：
- 聚焦于"单被试内部"的结构-动力学对应关系（控制了个体差异）
- 系统性的合成 connectome 扰动实验（不依赖多被试）
- 将结论定位为"原则性证明"（proof of concept）而非流行病学研究

**问题 3：GNN 的 Jacobian 是否反映了真实神经动力学的 Jacobian？**

神经网络的内部 Jacobian 不一定等于真实神经元集群动力学的 Jacobian。

**化解策略**：
- 用 Wilson-Cowan 模型（有明确的解析 Jacobian）作为基准验证
- 对 Jacobian 分析的结论保持保守，避免过度解读

#### 13.6.2 研究方向的难度排序

| 方向 | 难度 | 影响力 | 推荐优先级 |
|------|-----|-------|----------|
| 方向一：动力学压缩原则 | ★★★ | ★★★★ | 🥇 首先 |
| 方向二：结构必然边缘混沌 | ★★★★ | ★★★★★ | 🥈 其次 |
| 方向三：连接组→吸引子拓扑 | ★★★★★ | ★★★★★ | 🥉 最后 |

**建议**：先全力完成方向一，获得一个"干净"的机制解释结果，
以此为核心形成一篇 *Nature Neuroscience* 级别的工作。
方向二和三作为延伸工作或独立后续论文。

### 13.7 即时可行的分析扩展（优先级排序）

基于现有 `twinbrain-dynamics` 模块，以下扩展工作**无需额外数据收集**，
可立即启动：

#### 优先级 1：连接组谱分析模块（新增 `analysis/spectral_analysis.py`）

```python
# 关键功能
compute_spectral_dimension(C)          # 连接矩阵谱有效维度（参与率）
compute_spectral_eigenmodes(C, k)      # 前 k 个特征向量（主导模式）
align_jacobian_to_connectome(J, C)     # Jacobian 主模态 vs connectome 特征向量对齐角
```

**输入**：DTI 连接矩阵 $C$（已在图缓存中）  
**输出**：谱维度 $d_{\text{spec}}$，主模态矩阵，对齐角矩阵

#### 优先级 2：动力学维度估计整合（扩展 `analysis/embedding_dimension.py`）

当前模块已有 FNN 和关联维度 $D_2$。需要添加：
- **PCA 有效维度**：通过 PCA 解释方差达到 90% 的主成分数（快速初步估计）
- **内在维度估计**（TwoNN 方法，Facco et al. 2017 *Sci Rep*）：无需指定 $d$ 的无参数估计

#### 优先级 3：合成连接组扰动实验（扩展 `experiments/`）

新增 `experiments/connectome_perturbation.py`：
- 降秩截断、度保持随机化、选择性枢纽移除
- 批量运行自由动力学并收集维度、LLE、吸引子数量
- 输出：扰动量 vs 动力学指标的散点图（方向一的核心图）

#### 优先级 4：跨吸引子刺激实验（整合现有模块）

当前 `controllability.py` 推荐最优靶点，但没有验证这些靶点是否真的能
诱导吸引子跳转。整合步骤：
1. `attractor_analysis.py` → 吸引子质心 $\{A_k\}$
2. `controllability.py` → 最优靶点 $i^*$（针对每对 $(A_p, A_q)$）
3. `virtual_stimulation.py` → 从 $A_p$ 出发施加刺激，验证是否收敛到 $A_q$
