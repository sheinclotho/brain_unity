# TwinBrain 使用说明

> 面向普通用户的简单使用指南。不需要任何编程知识。

---

## 这个软件是做什么的？

TwinBrain 可以让你在屏幕上看到大脑的三维活动图像，支持模拟"刺激"大脑的某些区域，观察会发生什么变化。就像玩游戏一样直观。

---

## 快速开始（3步）

### 第一步：安装 Python

如果你还没有安装 Python，请前往 [python.org](https://www.python.org/downloads/) 下载安装（选择 3.8 或更高版本）。

### 第二步：下载并安装本软件

打开命令行（Windows 按 `Win+R` 输入 `cmd`，Mac 打开"终端"），输入以下命令：

```bash
git clone https://github.com/sheinclotho/brain_unity.git
cd brain_unity
pip install -r requirements.txt
```

### 第三步：启动！

```bash
python start.py
```

就这样！软件会自动启动，并在浏览器中打开脑可视化界面。

---

## 界面使用说明

启动后，浏览器会自动打开大脑可视化界面。

| 操作 | 效果 |
|------|------|
| **点击脑区圆圈** | 选中该脑区（会高亮显示） |
| **点击"施加刺激"** | 模拟刺激选中的脑区，观察活动变化 |
| **点击"重置"** | 恢复大脑初始状态 |
| **滑动"强度"** | 调整刺激强度 |

---

## 高级用法（可选）

如果你有训练好的脑模型文件（`.pt` 格式），可以用它来获得更真实的预测：

```bash
python start.py --model 你的模型文件路径.pt
```

如果你想使用 Unity 3D 进行更专业的可视化（需要安装 Unity），请参考 `项目规范说明书.md` 中的 Unity 集成章节。

---

## 常见问题

**Q：启动时提示找不到某个包怎么办？**  
A：再运行一次 `pip install -r requirements.txt`，然后重试。

**Q：浏览器没有自动打开怎么办？**  
A：手动打开浏览器，访问 `http://localhost:8765`

**Q：能在手机上用吗？**  
A：可以！在同一 Wi-Fi 下，手机浏览器访问电脑的 IP 地址加端口（如 `http://192.168.1.100:8765`）。

**Q：这个软件安全吗？会联网吗？**  
A：完全本地运行，不联网，不上传任何数据。
