# TwinBrain 使用说明

> 面向普通用户的简单使用指南。

---

## 这个软件是做什么的？

TwinBrain 可以实时可视化**200个皮质脑区的神经活动**，支持：

- 虚拟刺激仿真（选中脑区 → 施加刺激 → 观察传播）
- 时间轴回放（真实 fMRI 时序 / 刺激结果）
- 从训练缓存 `.pt` 文件加载真实数据

TwinBrain 提供**两种前端**，适合不同场景：

| 前端 | 优势 | 适合场景 |
|------|------|----------|
| **Web 前端** | 零安装，打开浏览器即用 | 快速演示、远程查看 |
| **Unity 前端** | 高质量渲染、VR 支持、可扩展 | 科研展示、深度交互 |

两个前端连接的是**同一个 Python 后端**，功能完全相同。

---

## 通用启动步骤（必做）

### 1. 安装 Python 依赖

```bash
git clone https://github.com/sheinclotho/brain_unity.git
cd brain_unity
pip install -r requirements.txt
```

### 2. 启动后端服务器

```bash
python start.py
```

输出 `Starting WebSocket server on 127.0.0.1:8765` 表示成功。保持此终端窗口运行。

可选：指定已训练模型：
```bash
python start.py --model path/to/hetero_gnn_trained.pt
```

## 方案 A — Web 前端（推荐首次使用）

启动后端后，用浏览器打开：

```
web_frontend/index.html
```

（直接双击文件，或在终端运行 `python start.py` 后访问 `http://localhost:8766`）

> **注意**：首次打开需要网络连接以加载 Three.js（来自 cdnjs.cloudflare.com）。

### Web 前端操作

| 操作 | 效果 |
|------|------|
| 鼠标拖动 | 旋转 3D 脑图 |
| 滚轮 | 缩放 |
| 点击脑区球体 | 选中（高亮放大） |
| ⚡ 施加刺激 | 发送刺激请求到后端，播放 60 帧传播动画 |
| ↺ 重置 | 清除选中，停止播放 |
| ▶ / ⏸ | 播放 / 暂停时间轴 |
| 拖动时间轴 | 跳到指定帧 |
| 📂 加载 .pt 文件 | 从训练缓存加载真实 fMRI 时序 |

## 方案 B — Unity 前端

### 前提条件

- Unity Hub + Unity **2020.3 LTS** 或更高版本
- 已安装 **NativeWebSocket** 包（见下方）

### 步骤 1：安装 NativeWebSocket

编辑 Unity 项目中的 `Packages/manifest.json`，在 `"dependencies"` 内添加一行：

```json
"com.endel.nativewebsocket": "https://github.com/endel/NativeWebSocket.git#upm",
```

保存后切回 Unity，等待自动导入。

或在 Unity 中：**Window → Package Manager → [+] → Add package from git URL**，输入：
```
https://github.com/endel/NativeWebSocket.git#upm
```

### 步骤 2：初始化 Unity 项目（一次性）

```bash
python unity_one_click_install.py --unity-project /path/to/your/UnityProject
```

可选：提供 FreeSurfer 数据以生成真实脑表面 OBJ：
```bash
python unity_one_click_install.py \
    --unity-project /path/to/UnityProject \
    --freesurfer-dir /path/to/freesurfer
```

### 步骤 3：在 Unity 中设置场景

1. 打开 Unity，菜单：**TwinBrain → Setup Scene**
2. 确认 WebSocket URL 为 `ws://127.0.0.1:8765`
3. 点击 **⚡ Build Scene**
4. 一键生成：BrainManager、200 个脑区球体、摄像机、UI 面板

### 步骤 4：运行

1. 确认后端已启动（`python start.py`）
2. 在 Unity 中点击 **Play ▶**
3. 脑区自动显示演示活动；连接后端后切换为实时数据

### Unity 操作

| 操作 | 效果 |
|------|------|
| 左键拖动 | 旋转摄像机（轨道模式） |
| 滚轮 | 缩放 |
| 点击脑区 | 选中/取消选中 |
| ⚡ 施加刺激 | 发送刺激，时间轴播放结果帧 |
| ▶ / ⏸ | 播放 / 暂停时间轴 |
| 时间轴滑块 | 拖动到指定帧 |
| 📂 加载缓存 | 输入 .pt 路径（留空=自动检测） |

## 加载真实 fMRI 数据

训练完成后会生成 `hetero_graphs.pt` 或 `eeg_data.pt`。

- **Web 前端**：右侧面板 → "加载缓存数据" → 输入路径（留空=自动检测）→ 点击 📂
- **Unity 前端**：StimulationPanel → "加载缓存" → 输入路径 → 点击按钮

服务器读取 `fmri.x_seq`（N×T×F），归一化为 [0,1]，推送给前端时间轴自动播放。

## 常见问题

**Q：启动时提示找不到某个包？**  
A：运行 `pip install -r requirements.txt`，然后重试。

**Q：Unity 提示 NativeWebSocket 未找到？**  
A：按"方案 B 步骤 1"安装 NativeWebSocket。

**Q：浏览器显示"演示模式（后端未连接）"？**  
A：确认 `python start.py` 正在运行，并且没有防火墙阻止 8765 端口。

**Q：Web 和 Unity 前端能同时连接吗？**  
A：可以！后端支持多客户端并发连接。
