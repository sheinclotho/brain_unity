# TwinBrain 使用说明

> 面向普通用户的简单使用指南。不需要任何编程知识。

---

## 这个软件是做什么的？

TwinBrain 可以让你在浏览器里看到**三维大脑活动图**。200 个皮质区域以球体形式分布在脑形轮廓上，颜色从蓝（低活动）到红（高活动）实时变化。支持：

- 鼠标拖动旋转脑图，滚轮缩放
- 点击脑区施加虚拟刺激，观察扩散传播
- 时间轴拖动回放完整时序数据
- 加载训练生成的 `.pt` 缓存文件，查看真实 fMRI 时序

---

## 快速开始（3步）

### 第一步：安装 Python

前往 [python.org](https://www.python.org/downloads/) 下载安装（3.8 或更高版本）。

### 第二步：安装依赖

```bash
git clone https://github.com/sheinclotho/brain_unity.git
cd brain_unity
pip install -r requirements.txt
```

### 第三步：启动！

```bash
python start.py
```

软件会自动启动后端，并在浏览器中打开 3D 可视化界面。

> **注意**：首次打开需要网络连接以加载 Three.js（约 600 KB，来自 cdnjs.cloudflare.com）。之后可离线使用。

---

## 界面操作说明

| 操作 | 效果 |
|------|------|
| **鼠标拖动** | 旋转 3D 脑图 |
| **滚轮** | 缩放 |
| **点击脑区球体** | 选中（白色轮廓高亮） |
| **⚡ 施加刺激** | 对选中区域施加虚拟刺激，展示传播效果（60帧时序） |
| **↺ 重置** | 清除选中，停止播放 |
| **▶ / ⏸** | 播放 / 暂停时间轴 |
| **拖动时间轴** | 跳到指定帧 |

---

## 加载真实数据

如果你运行过训练（`python main.py train ...`），会生成 `hetero_graphs.pt` 或 `eeg_data.pt` 文件。

1. 在界面右侧"加载缓存数据"处输入文件完整路径（可留空，自动搜索）
2. 点击 **📂 加载 .pt 文件**
3. 时间轴自动填充真实 fMRI 时序，拖动即可观察

---

## 高级用法（可选）

用训练好的脑模型运行（获得更真实的刺激预测）：

```bash
python start.py --model 你的模型文件路径.pt
```

## 常见问题

**Q：启动时提示找不到某个包怎么办？**  
A：再运行一次 `pip install -r requirements.txt`，然后重试。

**Q：浏览器没有自动打开怎么办？**  
A：手动打开浏览器，访问 `http://localhost:8765`

**Q：能在手机上用吗？**  
A：可以！在同一 Wi-Fi 下，手机浏览器访问电脑的 IP 地址加端口（如 `http://192.168.1.100:8765`）。

**Q：这个软件安全吗？会联网吗？**  
A：完全本地运行，不联网，不上传任何数据。
